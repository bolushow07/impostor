Configuración — solo una vez

Necesitas una base de datos gratuita de Supabase. Puedes entrar con tu cuenta de GitHub.

1
Crea cuenta y proyecto en Supabase
Ve a supabase.com → Start for free → inicia sesión con GitHub → New project → ponle un nombre → escribe cualquier contraseña → Create new project.
2
Crea la tabla de salas
En el menú lateral ve a SQL Editor → pulsa New query → pega este código y pulsa Run:

create table if not exists rooms ( code text primary key, data jsonb not null, updated_at timestamptz default now() ); alter table rooms enable row level security; create policy "public" on rooms for all using (true) with check (true);
3
Copia tus claves
En el menú lateral ve a Project Settings (icono engranaje) → API → copia la Project URL y la anon public key.
