Configuración inicial

Para jugar online desde cualquier parte necesitas conectar tu propia base de datos gratuita de Firebase. Solo hay que hacerlo una vez — los datos se guardan en el navegador.

1
Crea un proyecto en Firebase
Ve a console.firebase.google.com → "Añadir proyecto" → pon un nombre → desactiva Analytics → Crear proyecto.
2
Activa Realtime Database
En el menú lateral: Compilación → Realtime Database → Crear base de datos → elige región → modo de prueba (test mode) → Listo.
3
Copia tu configuración
En Configuración del proyecto (⚙️) → Tus apps → Web (</>) → Registra la app → copia los valores del objeto firebaseConfig.
